== Description ==

O Plugin da Real Seguro Viagem permite que você adicione nossa ferramenta comparativa de seguro viagem - um hotsite - ao seu site wordpress e aumente seus lucros.

Através deste hotsite, seu cliente realiza de forma rápida a cotação e compra do Seguro Viagem, com a facilidade de comparar entre as principais seguradoras e planos do mercado.

Torne-se um afiliado ainda hoje:

1 - faça seu cadastro em nosso Programa de Afiliados em https://www.seguroviagem.srv.br/afiliados;
2 - Instale este plugin em seu blog Worpress;
3 - Acesse o WP-Admin em “Seguro Viagem” informe seu código de afiliado.


== Installation ==

Em seu painel WordPress,
1. Acesse: Plugins > Adicionar novo
2. Procure por: Real Seguro Viagem
3. Ative o Plugin da Real em sua página.


No Wordpress.org:
Faça download do “Real Seguro Viagem Afiliados”
Faça upload da pasta “Real Seguro Viagem Afiliados”' para o diretório '/ wp-content / plugins /', usando seu método favorito (ftp, sftp, scp, etc ...)
Ative o plugin em sua página de plugins.

Depois da configuração
Após instalado, o plugin Real Seguro Viagem Afiliados permite que seus clientes realizem a cotação e compra do seguro viagem em seu site de forma rápida e prática.


== Frequently Asked Questions ==

Não sou brasileiro, posso utilizar este plugin?
Este plugin é voltado apenas para brasileiro. Entretanto, caso possua CPF e uma conta bancária no Brasil, poderá utilizá-lo.


= Quais são os requisitos para utilizar o plugin? =

1. PHP versão 5.3 ou superior.
2. Um site com o WordPress versão 3.5 ou superior
